CSS3

Projeto simples para reunir diversos exemplos de aplicações básicas das CSSs "puras", ou seja, apenas o CSS e o HTML. Vou esse projeto na forma de uma documentação sobre tudo que eu aprender sobre CSS, posso começar de maneira simples, estruturando em tópicos e apresentado aplicações em tela com os códigos de exemplo. Elementos de código inline serão renderizados da seguinte forma: background-color: red;
